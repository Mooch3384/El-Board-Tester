# Board Tester Pro

A desktop application for testing electrical boards with Arduino.

## Features

- Real-time voltage, current, power monitoring
- Frequency and wavelength measurement
- Threshold alerts (high/low warnings)
- Test recording with database storage
- Export reports (HTML, CSV)
- Modern dark UI

## Requirements

- Python 3.8+
- Arduino Uno/Nano with CH340
- ACS712 current sensor
- Voltage divider circuit

## Installation

```bash
pip install -r requirements.txt
python main.py
